## Alltagsauto-Exclusive

In the **"Alltagsauto-Exclusive"** project, I used technologies such as **NextJS**, **TypeScript**, **JavaScript**, **Tailwind CSS**, **Figma**, **PostgreSQL**, and the **SCRUM** methodology. The main objective was to develop a web platform for a Swiss startup specializing in premium car rentals by applying modern frontend technologies, software architecture principles, and agile development practices.

To achieve this, I contributed to the development of a responsive interface using **NextJS** and **Tailwind CSS**, participated in API integrations, and collaborated with the development team following the **SCRUM** methodology, using German as the primary language throughout the project.

### Project Overview

**"Alltagsauto-Exclusive"** is a web platform developed for **Alltagsauto Group (Switzerland)**, focused exclusively on premium car rentals. The application works as an e-commerce-style catalog where only administrators can publish available vehicles, while customers can browse listings and request a vehicle by contacting the company via email or phone.

This project represented my first international professional experience, working remotely for three months as part of an agile development team. During this time, I strengthened both my technical skills and my professional communication in German within an IT environment.

You can view the deployed project here:

**https://alltagsautoexclusive.vercel.app/**

For the main **Alltagsauto Group** website, dedicated to used car sales, visit:

**https://alltagsauto.ch/de/**

### Key Features

- **Premium Vehicle Catalog**: Users can browse a selection of exclusive vehicles with detailed information about each model.

- **Content Management**: Only administrators have permission to create and manage vehicle listings available on the platform.

- **Responsive Design**: The platform was developed to provide a consistent user experience across mobile devices, tablets, and desktop computers.

### Technologies and Tools Used

- **NextJS**: Used to develop a modern, scalable, and optimized application by leveraging Server-Side Rendering (SSR).
- **TypeScript**: Implemented to ensure safe, strongly typed, and maintainable code.
- **JavaScript**: Used to develop dynamic functionalities and client-side interactions.
- **Tailwind CSS**: Used to build a modern, flexible, and fully responsive user interface.
- **Figma**: Used to follow the UI/UX designs defined during the project.
- **PostgreSQL**: Used as the database system to manage the platform's information.
- **SCRUM**: Agile methodology used for task planning, team collaboration, and project organization.
