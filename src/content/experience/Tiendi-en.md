## Tiendi: Your online store

In the **"Tiendi"** project, I used technologies such as **NextJS**, **TypeScript**, **JavaScript**, **Tailwind**, **PostgreSQL**, and **Figma**. The main objective was to develop an e-commerce platform specialized in selling computer products, applying modern frontend technologies and an architecture focused on performance and scalability.

To achieve this, I developed a responsive interface capable of adapting to different devices such as mobile phones, tablets, and desktop computers. I used **NextJS** because of its server-side rendering (SSR) capabilities, allowing improvements in application performance and search engine optimization (SEO).

### Project Overview

**"Tiendi"** is an e-commerce platform focused on selling computer products, where users can browse products, make purchases, and manage orders with detailed information such as date, ID, time, and other relevant details.

The project was developed to apply advanced frontend development concepts, interface design principles, and external service integration, creating a simple, modern, and optimized shopping experience for users.

### Key Features

- **Product Catalog and Management**: Users can browse available computer products within the platform and access detailed information about each item.

- **Purchase and Order Management**: The application allows users to manage purchases and generate orders containing specific information such as date, identifier, and transaction details.

- **Responsive Design**: The interface was designed to provide a consistent experience across mobile devices, tablets, and desktop computers through an adaptive layout.

- **NextJS Optimization**: The application uses server-side rendering (SSR) to improve performance, initial loading speed, and SEO optimization.

### Technologies and Tools Used

- **NextJS**: Used to develop a scalable and optimized application, taking advantage of server-side rendering capabilities and improved performance.
- **TypeScript**: Implemented to ensure safer, structured, and maintainable code throughout the development process.
- **JavaScript**: Used to develop dynamic functionalities and client-side interactions.
- **Tailwind CSS**: Used to create a modern, flexible, and responsive interface through an efficient styling system.
- **Figma**: Used to design the user interface and create UI/UX prototypes before implementation.
- **PostgreSQL**: Used as the database system due to its reliability and ability to handle complex queries and transactions.
