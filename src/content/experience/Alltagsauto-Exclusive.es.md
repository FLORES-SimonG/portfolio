## Alltagsauto-Exclusive
Alltagsauto-Exclusive es un proyecto de servicio de alquiler de coches premium desarrollado para la startup suiza Alltagsauto Group. Como parte de su expansión, necesitaban una plataforma web dedicada exclusivamente a coches de alta gama. Este proyecto implicó la creación de un sitio web de estilo e-commerce donde solo el administrador puede publicar las ofertas disponibles, y los usuarios pueden solicitar el producto contactando por correo electrónico o teléfono.

Puedes ver el proyecto desplegado aquí: [Alltagsauto-Exclusive](https://alltagsautoexclusive.vercel.app/).

![ExampleAlltagsauto.](/assets/all-2.jpg)

Para el sitio principal de Alltagsauto Group, que trata con coches de segunda mano, visita: [Alltagsauto](https://alltagsauto.ch/de/).

>Tecnologías y herramientas clave utilizadas:
- **NextJS**: Elegido por su rendimiento y escalabilidad, ideal para una plataforma especializada de alta gama.
- **TypeScript**: Aseguró un código robusto y mantenible, mejorando el proceso de desarrollo.
- **JavaScript**: Utilizado para funcionalidades dinámicas e interacciones del lado del cliente.
- **Tailwind**: Proporcionó un framework CSS moderno y flexible para un diseño responsivo y estilizado.
- **Figma**: Usado para diseño UI/UX, creando una interfaz fluida y atractiva.
- **PostgreSQL**: Empleado como base de datos por su fiabilidad y eficiencia en el manejo de consultas complejas y transacciones.
