## Tiendi: Tu tienda online

En el proyecto **"Tiendi"**, utilicé tecnologías como **NextJS**, **TypeScript**, **JavaScript**, **Tailwind**, **PostgreSQL** y **Figma**. El objetivo principal fue desarrollar una plataforma de comercio electrónico especializada en la venta de productos informáticos, aplicando tecnologías frontend modernas y una arquitectura orientada al rendimiento y escalabilidad.

Para ello, desarrollé una interfaz responsive capaz de adaptarse a diferentes dispositivos como teléfonos móviles, tablets y ordenadores de escritorio. Utilicé **NextJS** por sus capacidades de renderizado del lado del servidor (SSR), permitiendo mejorar el rendimiento de la aplicación y optimizar aspectos relacionados con el posicionamiento SEO.

### Descripción del proyecto

**"Tiendi"** es una plataforma de comercio electrónico enfocada en la venta de productos informáticos, donde los usuarios pueden explorar productos, realizar compras y gestionar pedidos con información detallada como fecha, identificador, hora y otros datos relevantes.

El proyecto fue desarrollado para aplicar conocimientos avanzados de desarrollo frontend, diseño de interfaces y conexión con servicios externos, creando una experiencia de compra sencilla, moderna y optimizada para el usuario.

### Características principales

- **Catálogo y gestión de productos**: Los usuarios pueden visualizar productos informáticos disponibles dentro de la plataforma y acceder a información detallada de cada artículo.

- **Proceso de compra y pedidos**: La aplicación permite gestionar compras y generar pedidos con información específica como fecha, identificador y detalles de la transacción.

- **Diseño responsive**: La interfaz fue diseñada para ofrecer una experiencia consistente en dispositivos móviles, tablets y ordenadores mediante un diseño adaptable.

- **Optimización con NextJS**: La aplicación utiliza renderizado del lado del servidor (SSR) para mejorar el rendimiento, la carga inicial y la optimización SEO del sitio.

### Tecnologías y herramientas utilizadas

- **NextJS**: Utilizado para desarrollar una aplicación escalable y optimizada, aprovechando sus capacidades de renderizado del lado del servidor y mejora del rendimiento.
- **TypeScript**: Implementado para garantizar un código más seguro, estructurado y mantenible durante el desarrollo.
- **JavaScript**: Utilizado para desarrollar funcionalidades dinámicas e interacciones del lado del cliente.
- **Tailwind CSS**: Empleado para crear una interfaz moderna, flexible y responsive mediante un sistema de estilos eficiente.
- **Figma**: Utilizado para diseñar la interfaz de usuario y crear prototipos UI/UX antes de la implementación.
- **PostgreSQL**: Utilizado como sistema de base de datos por su fiabilidad y capacidad para gestionar consultas complejas y transacciones.
