## Alltagsauto-Exclusive

En el proyecto **"Alltagsauto-Exclusive"**, utilicé tecnologías como **NextJS**, **TypeScript**, **JavaScript**, **Tailwind CSS**, **Figma**, **PostgreSQL** y la metodología **SCRUM**. El objetivo principal fue desarrollar una plataforma web para una startup suiza especializada en el alquiler de vehículos premium, aplicando tecnologías modernas de desarrollo frontend, principios de arquitectura de software y metodologías ágiles de trabajo.

Para ello, participé en el desarrollo de una interfaz responsive utilizando **NextJS** y **Tailwind CSS**, colaboré en la integración con diferentes APIs y trabajé junto al equipo siguiendo la metodología **SCRUM**, utilizando el alemán como idioma principal durante toda la comunicación del proyecto.

### Descripción del proyecto

**"Alltagsauto-Exclusive"** es una plataforma web desarrollada para **Alltagsauto Group (Suiza)**, enfocada exclusivamente en el alquiler de vehículos de alta gama. La aplicación funciona como un catálogo tipo e-commerce donde únicamente los administradores pueden publicar nuevos vehículos disponibles, mientras que los clientes pueden consultar la información y contactar directamente mediante correo electrónico o teléfono para solicitar el servicio.

Este proyecto representó mi primera experiencia profesional internacional trabajando de forma remota durante tres meses dentro de un equipo ágil, fortaleciendo tanto mis conocimientos técnicos como mis habilidades de comunicación en alemán dentro del entorno IT.


### Características principales

- **Catálogo de vehículos premium**: Los usuarios pueden explorar una selección de vehículos exclusivos con información detallada sobre cada modelo.

- **Gestión de publicaciones**: Solo los administradores tienen permisos para crear y administrar las publicaciones de los vehículos disponibles.

- **Diseño responsive**: La plataforma fue desarrollada para ofrecer una experiencia de usuario consistente en dispositivos móviles, tablets y ordenadores.

- **Integración con APIs**: Participé en la integración de diferentes servicios necesarios para el funcionamiento de la aplicación y la comunicación entre frontend y backend.

### Tecnologías y herramientas utilizadas

- **NextJS**: Utilizado para desarrollar una aplicación moderna, optimizada y escalable mediante Server-Side Rendering (SSR).
- **TypeScript**: Implementado para garantizar un código seguro, tipado y fácilmente mantenible.
- **JavaScript**: Utilizado para desarrollar funcionalidades dinámicas e interacciones del lado del cliente.
- **Tailwind CSS**: Empleado para construir una interfaz moderna, flexible y completamente responsive.
- **Figma**: Utilizado para seguir los diseños UI/UX definidos durante el desarrollo del proyecto.
- **PostgreSQL**: Utilizado como sistema de base de datos para gestionar la información de la plataforma.
- **SCRUM**: Metodología ágil utilizada para la organización del trabajo, planificación de tareas y colaboración continua con el equipo.
