## FilmoTK

En el proyecto **"FilmoTK"**, utilicé tecnologías como **JavaScript**, **HTML**, **CSS**, **Bootstrap** y **Express**. El objetivo principal fue desarrollar una aplicación utilizando tecnologías web nativas para fortalecer mis conocimientos del desarrollo sin frameworks. 

Para ello, implementé una interfaz responsive con **Bootstrap**, desarrollé la lógica de la aplicación con **JavaScript**, estructuré el contenido con **HTML** y **CSS**, y utilicé **Express** para crear el servidor y gestionar las peticiones entre el cliente y la aplicación.

### Descripción del proyecto

**"FilmoTK"** es una aplicación de gestión de películas que permite buscar, crear, editar y eliminar registros, además de realizar filtrados y búsquedas sobre grandes volúmenes de datos. El proyecto fue desarrollado con el objetivo de practicar la integración entre una aplicación web y una base de datos no relacional **(MongoDB)**, implementando operaciones **CRUD** completas y una experiencia de usuario sencilla e intuitiva.

### Características principales

- **Gestión completa de películas (CRUD)**: Los usuarios pueden agregar nuevas películas, editar información existente, eliminar registros y visualizar el catálogo actualizado en tiempo real.

- **Búsqueda y filtrado de datos**: mplementé un sistema de búsqueda que permite localizar películas rápidamente y filtrar resultados cuando la cantidad de registros es elevada.

- **Integración con MongoDB**: El sistema se conecta a una base de datos no relacional para almacenar y recuperar la información de las películas mediante peticiones gestionadas por **Express**.

### Tecnologías y herramientas utilizadas

- **JavaScript**: Utilizado para desarrollar la lógica principal de la aplicación y la interacción con el usuario.
- **HTML y CSS**: Empleados para estructurar y estilizar la interfaz de usuario.
- **Bootstrap**: Elegido para crear una interfaz moderna, responsive y consistente.
- **Express**: Utilizado para construir el servidor y gestionar las rutas y peticiones HTTP.
- **MongoDB**: Base de datos no relacional utilizada para almacenar la información de las películas de forma flexible y escalable.
