# Portfolio (Next.js)

Este proyecto fue migrado de Astro a **Next.js (App Router)** manteniendo los estilos y diseño visual del portfolio.

## Requisitos

- Node.js 18+
- npm

## Desarrollo

```bash
npm install
npm run dev
```

Abrir: http://localhost:3000

## Build y ejecución en producción

```bash
npm run build
npm run start
```

## Estructura principal

- `src/app/`: rutas y páginas con App Router
- `src/components/`: componentes React reutilizables
- `src/content/experience/`: contenido markdown de experiencia
- `public/assets/`: imágenes y assets estáticos
